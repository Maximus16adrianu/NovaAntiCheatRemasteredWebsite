Combat

  • Aim (A-X)
    Looks for aim that turns too cleanly or snaps too hard.
  • Bow Aim (A)
    Looks for bow aim that stays too clean or locks too fast.
  • Auto Clicker (A-T)
    Looks for click patterns that are too steady or too repeated.
  • Auto Block (A-F)
    Looks for blocking and hitting at times that should not happen together.
  • Kill Aura (A-G)
    Looks for attack patterns that do not fit normal player input.
  • Crystal Aura (A-C)
    Looks for crystal actions that are too fast or too clean.
  • Reach (A-B)
    Looks for hits from farther away than they should be.
  • Container Reach (A)
    Looks for opening containers from too far away.
  • HitBox (A)
    Checks if the hit really lines up with the target.
  • WTap (A)
    Looks for repeated sprint resets around hits.
  • AutoTotem (A)
    Looks for totem swaps that happen too fast.
  • Backtrack (A-B)
    Looks for hits on old target positions instead of the real one.
  • Combat Engine (A)
    Builds a score over time from how someone fights.
  • Crystal Engine (A)
    Builds a score over time from crystal fights.

Supported Versions

Version Support

Official support covers Minecraft 1.13 to 1.21.11. Version 26.1 is early-tested and experimental. Older versions down to 1.8 can work, but they may run with reduced protection when some checks are not compatible.

Server Software Tested on Paper and Bukkit

Other server software has not been fully tested yet, so Paper and Bukkit are still the main verified targets.

Officially supported 1.13 to 1.21.11

Main tested range for current Java servers.

Experimental 26.1

Early-tested, usable, and still not fully guaranteed.

Legacy best effort 1.8 to 1.12

Can work with fewer checks when incompatible checks are disabled automatically.

Modern target

Modern Java servers are the main supported target for Nova.

Legacy behavior

Nova does not claim full real-server 1.8 support because that range is not the main tested target.

Compatibility mode

Incompatible checks are disabled automatically, so protection can be lighter on older setups.
