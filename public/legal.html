Privacy / License Authentication Data

Nova AntiCheat uses an online license and authentication backend to validate licenses, prevent abuse, detect unauthorized redistribution and keep licensed installations secure.

During activation, heartbeat checks and shutdown notifications, the plugin may transmit the following data to the Nova backend:

  • license key and configured username
  • server name / instance name
  • plugin version
  • server IP address and server port
  • host name / PC name
  • operating system name, version and architecture
  • local system user name
  • machine identifier or generated stable machine identity
  • processor/CPU identifier and available processor count
  • generated server instance UUID
  • hashed server path identifiers, such as server root path hash, world container path hash and plugins path hash
  • session token, heartbeat status and shutdown reason

The data is used only for license validation, device/session management, abuse prevention, security checks, support and service reliability.

License authentication traffic is protected using encrypted transport. However, the backend must process the submitted data to validate the license and operate the service.

Authentication and license-related records may be retained for as long as necessary to operate the license system, prevent abuse, investigate unauthorized redistribution and comply with legal or business requirements.
